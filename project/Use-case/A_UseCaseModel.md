# 유스케이스 모델

개발할 시스템의 기능이 무엇인지 식별하는 작업.

## 액터

시스템을 사용하는 사람, 프로세스, 장치를 가리킨다.

- 주액터

  - 비회원
  - 회원
    - 팀원
      - 팀장

- 보조액터
  - 다음주소 검색 시스템
  - 페이스북 로그인
  - 카카오로그인

## 유스케이스

액터가 시스템을 통해 수행하는 업무이다.

- 비회원

  - 프로젝트 조회
  - 게시글 조회

- 회원

  - 프로젝트 생성(Create Project) 동사+목적어
  - 프로젝트 참여, 프로젝트 조회
  - 게시글 등록, 게시글 조회, 게시글 변경, 게시글 삭제

- 팀원
- 작업 조회, 작업 변경

- 팀장
  - 작업 등록, 작업 삭제
  - 프로젝트 변경, 프로젝트 조회, 프로젝트 삭제
